**Glossary for this plan:**
- *Null-move pruning:* skip a turn; if the opponent STILL can't get below beta even moving twice in a row, the node is so good it's prunable. Guards: never in check, never twice in a row, never in pawn-only endgames (zugzwang), only when static eval already ≥ beta.
- *LMR:* late, quiet moves (bad ordering rank) get searched at reduced depth; surprises get re-searched at full depth.
- *Aspiration:* start each ID iteration with a narrow window around the last score; widen exponentially on fail.
- *RFP (reverse futility):* if static eval beats beta by a depth-scaled margin at a shallow node, trust it and prune.
- *Futility:* at shallow depth with hopeless static eval, skip quiet moves entirely (a quiet move won't recover 300cp).
- *improving:* static eval better than two plies ago (same side) — confidence signal that scales margins.
- *Texel tuning:* fit eval parameters so sigmoid(K·eval) predicts game results over a labeled position set; K is fit once then frozen; analytic gradients because the eval is linear in the parameters.

```rust
    /// Null move: pass the turn (search-only device; illegal in chess).
    /// Pairs strictly with unmake_null — NEVER with unmake().
    pub fn make_null(&mut self) {
        self.undo_stack.push(Undo {
            mv: Move::NULL,
            captured: None,
            castling: self.castling,
            ep: self.ep,
            halfmove: self.halfmove,
            key: self.key,
        });
        self.key_history.push(self.key);
        if let Some(ep) = self.ep.take() {
            self.key ^= KEYS.ep_file[ep.file() as usize];
        }
        self.halfmove += 1;
        self.stm = self.stm.flip();
        self.key ^= KEYS.black_to_move;
        debug_assert_eq!(self.key, self.compute_key());
    }

    pub fn unmake_null(&mut self) {
        self.key_history.pop();
        let u = self.undo_stack.pop().expect("unmake_null: empty undo stack");
        debug_assert_eq!(u.mv, Move::NULL, "unmake_null paired with a real make");
        self.stm = self.stm.flip();
        self.castling = u.castling;
        self.ep = u.ep;
        self.halfmove = u.halfmove;
        self.key = u.key;
    }

    /// Anything beyond king+pawns for `color` (zugzwang guard for null-move).
    pub fn has_non_pawn_material(&self, color: Color) -> bool {
        (self.piece_bb(color, PieceType::Knight)
            | self.piece_bb(color, PieceType::Bishop)
            | self.piece_bb(color, PieceType::Rook)
            | self.piece_bb(color, PieceType::Queen))
        .any()
    }
```

- [ ] **Step 2.1: The pruning block.** In `negamax`, immediately after the static-eval plumbing from T1, insert:

```rust
        // null-move pruning: if we pass the turn and the opponent STILL
        // can't get under beta, this node is prunable. Guards: in check
        // (illegal), consecutive nulls (infinite recursion), pawn-only
        // material (zugzwang), eval below beta (no margin to give away).
        if ply > 0
            && !in_check
            && depth >= 3
            && static_eval >= beta
            && self.stack[ply - 1].current_move != Move::NULL
            && self.pos.has_non_pawn_material(self.pos.stm())
        {
            const R: i32 = 3;
            self.stack[ply].current_move = Move::NULL;
            self.pos.make_null();
            let score = -self.negamax(depth - 1 - R, -beta, -beta + 1, ply + 1);
            self.pos.unmake_null();
            if self.stopped {
                return 0;
            }
            if score >= beta {
                // never return unproven mate scores from a null search
                return if score >= MATE_BOUND { beta } else { score };
            }
        }
```

(NOTE: `depth - 1 - R` can go ≤ 0 — negamax's existing `depth <= 0` dispatch handles it. The Evaluator hooks are deliberately NOT called around null — see plan header.)

```rust
    /// Narrow window around the previous score; widen exponentially on fail.
    fn aspiration(&mut self, depth: i32, guess: i32) -> (Option<Move>, i32) {
        let mut delta = 25;
        let mut alpha = (guess - delta).max(-INF);
        let mut beta = (guess + delta).min(INF);
        loop {
            let (mv, score) = self.search_root(depth, alpha, beta);
            if self.stopped {
                return (mv, score);
            }
            if score <= alpha {
                // fail-low: lower alpha, pull beta toward the fail point
                beta = (alpha + beta) / 2;
                alpha = (score - delta).max(-INF);
            } else if score >= beta {
                beta = (score + delta).min(INF);
            } else {
                return (mv, score);
            }
            delta *= 2; // mate-region fails widen to the full window fast
        }
    }
```

```rust
//! Texel tuner (spec §6.3): fits material (N,B,R,Q; P=100 and K=0 anchored)
//! and the six single-phase PSTs to game results over quiet-labeled.epd.
//! Loss: MSE(result - sigmoid(K * eval / 400)); K fit once on the warm-start
//! params, then frozen. Optimizer: Adam, full-batch analytic gradients (the
//! eval is linear in every parameter). Emits a regenerated src/eval/psqt.rs
//! on stdout:
//!   cargo run --release --bin tune -- tools/data/quiet-labeled.epd > src/eval/psqt.rs
//! Optional args: [epochs=400] [lr=0.05] [limit=0(all)]

use nebchess::board::{Color, PieceType, Position};
use nebchess::eval::psqt::{MATERIAL, TABLES};

const N_MATERIAL: usize = 4; // N, B, R, Q (P anchored 100, K anchored 0)
const N_PARAMS: usize = N_MATERIAL + 6 * 64;

struct Sample {
    features: Vec<(u16, i8)>, // (param index, +1 white / -1 black)
    pawn_diff: i32,           // anchored pawn material (not a parameter)
    result: f64,              // 1.0 / 0.5 / 0.0 white-relative
}

fn extract(pos: &Position, result: f64) -> Sample {
    let mut features = Vec::with_capacity(32);
    let mut pawn_diff = 0i32;
    for pt in PieceType::ALL {
        for sq in pos.piece_bb(Color::White, pt) {
            features.push(((N_MATERIAL + pt.index() * 64 + (sq.index() ^ 56)) as u16, 1));
            match pt {
                PieceType::Pawn => pawn_diff += 100,
                PieceType::King => {}
                _ => features.push(((pt.index() - 1) as u16, 1)),
            }
        }
        for sq in pos.piece_bb(Color::Black, pt) {
            features.push(((N_MATERIAL + pt.index() * 64 + sq.index()) as u16, -1));
            match pt {
                PieceType::Pawn => pawn_diff -= 100,
                PieceType::King => {}
                _ => features.push(((pt.index() - 1) as u16, -1)),
            }
        }
    }
    Sample { features, pawn_diff, result }
}

fn eval(s: &Sample, params: &[f64; N_PARAMS]) -> f64 {
    let mut e = s.pawn_diff as f64;
    for &(idx, sign) in &s.features {
        e += params[idx as usize] * sign as f64;
    }
    e
}

fn sigmoid(k: f64, e: f64) -> f64 {
    1.0 / (1.0 + 10f64.powf(-k * e / 400.0))
}

fn mse(samples: &[Sample], params: &[f64; N_PARAMS], k: f64) -> f64 {
    samples
        .iter()
        .map(|s| {
            let d = s.result - sigmoid(k, eval(s, params));
            d * d
        })
        .sum::<f64>()
        / samples.len() as f64
}

fn warm_start() -> [f64; N_PARAMS] {
    let mut p = [0f64; N_PARAMS];
    // material: indices 0..4 = N,B,R,Q from the current tables
    for (i, pt) in [PieceType::Knight, PieceType::Bishop, PieceType::Rook, PieceType::Queen]
        .iter()
        .enumerate()
    {
        p[i] = MATERIAL[pt.index()] as f64;
    }
    for pt in 0..6 {
        for sq in 0..64 {
            p[N_MATERIAL + pt * 64 + sq] = TABLES[pt][sq] as f64;
        }
    }
    p
}

fn emit(params: &[f64; N_PARAMS], k: f64, train_mse: f64, val_mse: f64) {
    let names = ["PAWN", "KNIGHT", "BISHOP", "ROOK", "QUEEN", "KING"];
    println!("//! GENERATED by `cargo run --release --bin tune`. Do not edit.");
    println!("//! Texel-tuned on quiet-labeled.epd (725k, MIT). K = {k:.3},");
    println!("//! train MSE = {train_mse:.6}, validation MSE = {val_mse:.6}.");
    println!("//! Layout: rank-8 row first; white reads PST[sq ^ 56], black PST[sq].");
    println!();
    let m = |i: usize| params[i].round() as i32;
    println!(
        "pub const MATERIAL: [i32; 6] = [100, {}, {}, {}, {}, 0]; // P N B R Q K",
        m(0), m(1), m(2), m(3)
    );
    for (pt, name) in names.iter().enumerate() {
        println!();
        println!("#[rustfmt::skip]");
        println!("pub const {name}: [i32; 64] = [");
        for row in 0..8 {
            let cells: Vec<String> = (0..8)
                .map(|col| format!("{:>4},", params[N_MATERIAL + pt * 64 + row * 8 + col].round() as i32))
                .collect();
            println!("    {}", cells.join(""));
        }
        println!("];");
    }
    println!();
    println!("pub const TABLES: [&[i32; 64]; 6] = [&PAWN, &KNIGHT, &BISHOP, &ROOK, &QUEEN, &KING];");
}

fn main() {
    let args: Vec<String> = std::env::args().skip(1).collect();
    let path = args.first().expect("usage: tune <epd> [epochs] [lr] [limit]");
    let epochs: usize = args.get(1).and_then(|s| s.parse().ok()).unwrap_or(400);
    let lr: f64 = args.get(2).and_then(|s| s.parse().ok()).unwrap_or(0.05);
    let limit: usize = args.get(3).and_then(|s| s.parse().ok()).unwrap_or(0);

    let data = std::fs::read_to_string(path).expect("read epd");
    let mut samples = Vec::new();
    for line in data.lines() {
        // format: <placement> <stm> <castling> <ep> c9 "1-0"; (counters absent)
        let Some((fen4, tail)) = line.split_once(" c9 ") else { continue };
        let result = if tail.contains("1-0") {
            1.0
        } else if tail.contains("0-1") {
            0.0
        } else {
            0.5
        };
        let Ok(pos) = Position::from_fen(&format!("{fen4} 0 1")) else { continue };
        samples.push(extract(&pos, result));
        if limit > 0 && samples.len() >= limit {
            break;
        }
    }
    eprintln!("loaded {} samples", samples.len());
    let split = samples.len() * 9 / 10;
    let (train, val) = samples.split_at(split);

    let mut params = warm_start();

    // fit K once on the warm-start params (coarse-to-fine line search)
    let mut k = 1.0;
    let mut step = 0.5;
    for _ in 0..12 {
        let (lo, hi) = (mse(train, &params, k - step), mse(train, &params, k + step));
        let mid = mse(train, &params, k);
        if lo < mid && lo <= hi {
            k -= step;
        } else if hi < mid {
            k += step;
        } else {
            step /= 2.0;
        }
    }
    eprintln!("fitted K = {k:.4}, initial train MSE = {:.6}", mse(train, &params, k));

    // Adam, full batch
    let (mut m, mut v) = ([0f64; N_PARAMS], [0f64; N_PARAMS]);
    let (b1, b2, eps) = (0.9, 0.999, 1e-8);
    let scale = k * f64::ln(10.0) / 400.0; // d/dx sigmoid10(kx/400) factor
    for epoch in 1..=epochs {
        let mut grad = [0f64; N_PARAMS];
        for s in train {
            let p = sigmoid(k, eval(s, &params));
            // d(MSE)/d(param) = -2 (r - p) p(1-p) scale * sign
            let common = -2.0 * (s.result - p) * p * (1.0 - p) * scale;
            for &(idx, sign) in &s.features {
                grad[idx as usize] += common * sign as f64;
            }
        }
        let n = train.len() as f64;
        for i in 0..N_PARAMS {
            let g = grad[i] / n;
            m[i] = b1 * m[i] + (1.0 - b1) * g;
            v[i] = b2 * v[i] + (1.0 - b2) * g * g;
            let mh = m[i] / (1.0 - b1.powi(epoch as i32));
            let vh = v[i] / (1.0 - b2.powi(epoch as i32));
            params[i] -= lr * 100.0 * mh / (vh.sqrt() + eps); // cp-scale lr
        }
        if epoch % 50 == 0 {
            eprintln!(
                "epoch {epoch}: train {:.6} val {:.6}",
                mse(train, &params, k),
                mse(val, &params, k)
            );
        }
    }
    emit(&params, k, mse(train, &params, k), mse(val, &params, k));
}
```
